# PSPDFKit for iOS - Minimal

This is the Minimal example app for [PSPDFKit for iOS](https://pspdfkit.com/pdf-sdk/ios/). This project shows how to perform a minimal PSPDFKit integration in a basic iOS application. It also contains code for simple use cases of [Processor](https://pspdfkit.com/api/ios/Classes/PSPDFProcessor.html) and [PDFLibrary](https://pspdfkit.com/api/ios/Classes/PSPDFLibrary.html). The example is written in Swift.

## Requirements

* The latest [Xcode](https://developer.apple.com/xcode/).

## Getting Started

* Click on the green "Code" button and select the "Open with Xcode" option.<sup>[1](#footnote1)</sup>
* Follow the steps in Xcode to clone and open the project.
* Build and run.
